- [angularjs](https://github.com/angular/angular.js)
- [toastr](https://github.com/CodeSeven/toastr)

## Installation
Sanji logger service is based on es6 + webpack to development and embrace npm to
install it.

```shell
npm install sanji-logger-ui --save
```
